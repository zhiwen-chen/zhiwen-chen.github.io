Python-math模块

发表于 2019-11-17 分类于 Python基础教程

本篇正文

  • 引入math模块

  • 常用数学函数

  1. ceil:取大于等于x的最小的整数值，如果x是一个整数，则返回x

  2. copysign:把y的正负号加到x前面，可以使用0

  3. cos:求x的余弦，x必须是弧度

  4. degrees:把x从弧度转换成角度

  5. e:表示一个常量

  6. exp:返回math.e,也就是2.71828的x次方

  7. expm1:返回math.e的x(其值为2.71828)次方的值减１

  8. fabs:返回x的绝对值

  9. factorial:取x的阶乘的值

  10. floor:取小于等于x的最大的整数值，如果x是一个整数，则返回自身

  11. fmod:得到x/y的余数，其值是一个浮点数

  12. frexp:返回一个元组(m,e),其计算方式为：x分别除0.5和1,得到一个值的范围

  13. fsum:对迭代器里的每个元素进行求和操作

  14. gcd:返回x和y的最大公约数

  15. hypot:如果x是不是无穷大的数字,则返回True,否则返回False

  16. isfinite:如果x是正无穷大或负无穷大，则返回True,否则返回False

  17. isinf:如果x是正无穷大或负无穷大，则返回True,否则返回False

  18. isnan:如果x不是数字True,否则返回False

  19. ldexp:返回x*(2**i)的值

  20. log:返回x的自然对数，默认以e为基数，base参数给定时，将x的对数返回给定的base,计算式为：log(x)/log(base)

  21. log10:返回x的以10为底的对数

  22. log1p:返回x+1的自然对数(基数为e)的值

  23. log2:返回x的基2对数

  24. modf:返回由x的小数部分和整数部分组成的元组

  25. pi:数字常量，圆周率

  26. pow:返回x的y次方，即x**y

  27. radians:把角度x转换成弧度

  28. sin:求x(x为弧度)的正弦值

  29. sqrt:求x的平方根

  30. tan:返回x(x为弧度)的正切值

  31. trunc:返回x的整数部分

Python输出

发表于 2019-11-14 更新于 2019-11-15 分类于 Python基础教程

本节内容：print()函数

  • 简单玩法：

    1. print(var1,var2)#直接输出变量
    2. print(“str”,var)#字符串后加变量
    3. print(“words %d %s “ %(num,str))#传说中的占位符

  • 普通玩法：

    1. 语法格式：print(*objects,sep=’’,end=’\n’[,file=sys.stdout,flush=False])#括号[]内的参数很少使用
    2. 参数说明：objects转为字符串的对象，然后写入file参数指定的文件，sys.stdout为标准输出;sep每个对象间分隔的参数，默认是空格；end为结尾参数，默认为换行。
      1.有趣的示例：
      1. print(‘Hello’,’world!’,sep=’**‘) 输出 Hello**world
      2. print()默认输出后换行，不想换行可以print(end=’’)修改结尾参数
      3. 使用转义字符控制输出格式如\t(8个空格的制表符) \n
      4. 使用转义字符输出特殊字符，\，\’，\”等
      5. 字符对齐：左对齐ljust(n)、右对齐rjust(n)和居中center(n),n为最大显示字符个数；如
        str = "this is string example....wow!!!";
print str.ljust(50);
      6. zifill()函数0左填充
        print str.zifill();

Python位运算符

发表于 2019-11-10 分类于 Python基础教程

位运算的规则：把数字转为二进制后进行运算，运算结果转为原数字类型

  • Python的位运算符
运算符 操作 实例
& 按位与；如果对应位同为1则结果为1，否则为0 12&5 结果 4
I 按位或；如果对应位有一个为1则结果为1，否者为0 12I5 结果 13
^ 按位异或；如果对应位不相等，结果为1，否者为0 12^5 结果 9
~ 按位取反；对应位反转，如果原来为1(0)，变成0(1) ~12 结果 -13
<< 左移运算符；运算数的各二进制位左移若干位，高位丢弃，低位补零 12<<5 结果为384
>> 右移运算符；与上相反，低位丢弃，高位补零 12>>5 结果为0

Python基本数据类型之变量

发表于 2019-11-04 分类于 Python基础教程

  • 变量特点

    1. 变量不需要声明，但是必须赋值
    2. 变量命名：（下划线或字母）+（任意数目字母、数字或下划线）
  • 注意
    1. 变量不能以数字作为开头
    2. 系统关键字不能作为变量名使用，如if,while等
    3. 除了下划线其他字符不能使用
    4. 变量区分大小写
    5. 前后有下划线的变量名_X_为系统变量
    6. 变量是根据赋值所更改类型
    7. 赋值方法
      1. 普通赋值：y=1
      2. 链式赋值：y=x=a=1
      3. 多元赋值：x,y = 1,2 x,y=y,x
      4. 增量赋值：x+=1
