Python基本数据类型之变量

发表于 2019-11-04 分类于 Python基础教程

  • 变量特点

    1. 变量不需要声明，但是必须赋值
    2. 变量命名：（下划线或字母）+（任意数目字母、数字或下划线）
  • 注意
    1. 变量不能以数字作为开头
    2. 系统关键字不能作为变量名使用，如if,while等
    3. 除了下划线其他字符不能使用
    4. 变量区分大小写
    5. 前后有下划线的变量名_X_为系统变量
    6. 变量是根据赋值所更改类型
    7. 赋值方法
      1. 普通赋值：y=1
      2. 链式赋值：y=x=a=1
      3. 多元赋值：x,y = 1,2 x,y=y,x
      4. 增量赋值：x+=1

Python基本数据类型之字符串

发表于 2019-11-03 分类于 Python基础教程

常用” “或’ ‘或’’’ ‘’’括起来，同时使用反斜杠\转义特殊字符的一段文字。

  • 注意
    1. 字符串前加r，反斜杠不转义；如r'\t\n\u'
    2. 连接字符串，str1 + str2。字符串重复 str1 * num 。
    3. 续行符：\（后面不能有任何内容）；”””…”””或’’’…’’’续多行（没用过）
    4. 字符串索引方式：从左为0开始，从右边为-1开始到-n
    5. 字符串切片：[头下标:尾下标:步数]，[开始，结束)，前闭后开，步数不输入时，默认为1；例如str ="abcdefg" ，可以切除“abc”使用str[0:3]、str[0:3:1]，切出“cba”使用str[-5:-8]，切除“aceg”使用str[0:7:2]、str[0::2]。
    6. 字符串不能被改变：如word[0] = ‘m’会报错。
    7. find()函数：找子字符串，如str.find(sub_str)，找到为子字符串最左侧开始位置，找不到返回-1
    8. lower()全小写化，str.lower();upper()全大写,str.upper()。
    9. replace()，替换字符，如str.replace(old,new)。

Python基本数据类型之数值

发表于 2019-11-02 更新于 2019-11-03 分类于 Python基础教程

Python 支持数据自定义根据变化数据类型，但是更改类型其存储空间会重新分配。

  • 整型（int）Python3 整型可以做 long 类型使用。 1. 二进制：以 0b 开头。例如，0b11011 表示十进制的 27。 1. 八进制：以 0o 开头。例如，0o33 表示十进制的 27。 1. 十进制：正常显示。 1. 十六进制：以 0x 开头。例如，0x1b 表示十进制的 27。
  • 浮点型（float）
    例如 2.7 或 2.7e2=2.7*100
  • 布尔型（bool）
    0 为 Flase,其他常数为 True
  • 复数（complex）
    复数由实数部分和虚数部分组成，可以用 a+bj 表示，也可以使用 complex(a,b);实数与虚数部分都为浮点。
